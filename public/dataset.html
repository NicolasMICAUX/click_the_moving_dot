About the Dataset

This dataset contains mouse movement tracking data and game session information from the "Click the Moving Dot" game. Each record represents a single mouse position tracking event during gameplay.

Use Cases:

  • Training AI models to predict dot movement patterns
  • Analyzing user behavior and reaction times
  • Studying human-computer interaction patterns
  • Research on mouse tracking and motor skills
Note: The dataset is cached and updated once daily to ensure optimal performance. Downloads may include data up to 24 hours old.

Data Schema

Field Name Type Description
sessionUid String Unique identifier for the game session
userUid String Unique identifier for the user (persistent across sessions)
level Integer Game level when this tracking point was recorded
maxSpeed Float Maximum allowed speed for the dot at this level
sessionStartTime Timestamp Unix timestamp when the session started
sessionEndTime Timestamp Unix timestamp when the session ended
timestamp Timestamp Unix timestamp when this tracking point was recorded
dotX Float X coordinate of the dot (0-800 pixels)
dotY Float Y coordinate of the dot (0-800 pixels)
mouseX Float X coordinate of the mouse cursor (0-800 pixels)
mouseY Float Y coordinate of the mouse cursor (0-800 pixels)

Technical Notes

  • Coordinate System: Origin (0,0) is at the top-left corner of the game arena
  • Arena Size: 800x800 pixels
  • Sampling Rate: Mouse movements are tracked in real-time as they occur
  • Dot Radius: 15 pixels (useful for collision detection analysis)
  • Speed Progression: Each level increases max speed by 0.2 units
